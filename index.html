Успешная активация API

Следуйте инструкциям ниже для подключения моделей к вашим рабочим средам.

👾 Инструкция для Claude Code

Откройте терминал или командную строку.
Убедитесь, что у вас установлен Claude Code (команда npm install -g @anthropic-ai/claude-code).
Скопируйте конфигурацию ниже и вставьте её в файл settings.json (он находится в папке конфигурации Claude).
{
  "keys": {
    "anthropic_api_url": "https://api.anthropic.com",
    "anthropic_api_key": "Загрузка ключа...",
    "model_name": "claude-4.8-opus"
  }
}

☂️ Инструкция для VS Code (Roo Code / Cline / Continue)

Откройте настройки вашего расширения в VS Code.
В разделе API Provider выберите Anthropic.
Заполните поля данными из блока ниже:
API Provider: Anthropic
Base URL: https://api.anthropic.com
API Key: Загрузка ключа...
