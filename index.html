Твой доступ к API 🔮

Инструкция по настройке и списки моделей для комфортной работы.

⚡ 1. Автоматическая настройка (Самый быстрый способ)

Если ты используешь расширение Claude Code в VS Code, Cursor или Antigravity, просто скачай скрипт для своей платформы. Он настроит всё сам!

⚠️ Важно: перед запуском закрой редактор кода, чтобы настройки применились!

Для VS Code:

⬇️ Скачать .bat (Windows) 🍎 Скачать .command (Mac)

Для Cursor:

⬇️ Скачать .bat (Windows) 🍎 Скачать .command (Mac)

Как использовать: запусти скачанный файл, вставь свой выданный ключ (sk-...) и нажми Enter. Твой редактор откроется автоматически.

🟣 2. Ручная настройка Cline / Roo Code

Для плагинов вроде Cline или Roo Code выбери в настройках провайдера (API Provider) вариант OpenAI Compatible и введи эти данные:

Base URL: https://api.nghimmo.com/v1
API Key: Твой ключ sk-...
Model ID: Например, nghi/claude-sonnet-4.6

🛠 Решение проблем (Ошибка JSON)

Если в списке моделей пусто, или плагин выдает ошибку:
"Failed to set model: Unexpected end of JSON input"

  1. Открой настройки расширения Claude Code (иконка шестеренки).
  2. Найди раздел Environment Variables и нажми на синюю ссылку Edit in settings.json.
  3. Удали всё из открывшегося файла и вставь этот код:
{ "claudeCode.environmentVariables": { "ANTHROPIC_BASE_URL": "https://api.nghimmo.com", "ANTHROPIC_API_KEY": "sk-ТВОЙ_КУПЛЕННЫЙ_КЛЮЧ", "ANTHROPIC_MODEL": "nghi/claude-opus-4.8", "ANTHROPIC_SMALL_FAST_MODEL": "nghi/claude-haiku-4.5" } }

Вместо sk-ТВОЙ_КУПЛЕННЫЙ_КЛЮЧ аккуратно вставь свой ключ, сохрани файл и перезапусти редактор. Всё заработает! ⚡

👾 Доступные модели

Обрати внимание, что перед названием модели при ручном вводе всегда должна стоять приставка nghi/.

Самые мощные (Opus) Для сложных задач и глубокого анализа.
nghi/claude-opus-4.8
nghi/claude-opus-4.8-thinking
Оптимальные (Sonnet) Лучший баланс цены и качества. Для ежедневных задач.
nghi/claude-sonnet-4.6
nghi/claude-sonnet-4.6-thinking
Быстрые (Haiku) Для простых правок, опечаток и вопросов.
nghi/claude-haiku-4.5
