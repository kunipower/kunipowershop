1. ИНСТРУКЦИЯ ДЛЯ CURSOR

  • Откройте настройки Cursor (нажмите на шестеренку в правом верхнем углу).
  • Перейдите в раздел Models.
  • Отключите стандартные ползунки (Anthropic / OpenAI).
  • Найдите пункт OpenAI API Key (или Custom Provider).
  • Включите Override OpenAI Base URL и вставьте туда ссылку из блока ниже.
  • В поле API Key вставьте ваш скопированный ключ.
  • В списке моделей (Model names) нажмите + Add model и вручную впишите: claude-4.8-opus
Base URL: https://api.nghimmo.com/v1
API Key:  Загрузка...
Model:    claude-4.8-opus

2. РАСШИРЕНИЯ VS CODE (ROO CODE / CLINE)

  • Откройте расширение Roo Code или Cline в вашем VS Code.
  • Нажмите на иконку настроек (шестеренка) в окне расширения.
  • В самом первом поле API Provider выберите OpenAI Compatible (Именно OpenAI Compatible, это важно для правильной маршрутизации!).
  • Заполните оставшиеся поля точно так, как указано в блоке ниже:
API Provider: OpenAI Compatible
Base URL:     https://api.nghimmo.com/v1
API Key:      Загрузка...
Model ID:     claude-4.8-opus

3. CLAUDE CODE (ТЕРМИНАЛ / CLI)

  • Найдите конфигурационный файл на вашем компьютере:
    - Windows: %APPDATA%\Claude Code\settings.json
    - Mac / Linux: ~/.mcp/settings.json
  • Откройте этот файл через любой текстовый редактор (например, блокнот).
  • Полностью удалите всё, что там есть, и вставьте код ниже.
{
  "keys": {
    "anthropic_api_url": "https://api.nghimmo.com",
    "anthropic_api_key": "Загрузка...",
    "model_name": "claude-4.8-opus",
    "anthropic_default_model": "claude-4.8-opus",
    "anthropic_max_tokens": "8192",
    "anthropic_temperature": "0.5"
  }
}
